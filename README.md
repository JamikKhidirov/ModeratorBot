# 🤖 ModeratorBot

Мощный Telegram-бот на **aiogram 3** для модерации чатов, управления рекламой, автопостинга и администрирования.

## 🚀 Быстрый старт

```bash
# 1. Клонировать
git clone https://github.com/JamikKhidirov/ModeratorBot.git
cd ModeratorBot

# 2. Установить зависимости
pip install -r requirements.txt

# 3. Настроить .env
#    Открой файл .env и вставь токен от @BotFather:
#    BOT_TOKEN=1234567890:ABCdef...
#    ADMIN_IDS=твой_telegram_id    (форматы: 12345 или [12345, 67890])

# 4. Запустить
python app.py

# 5. Проверить конфиг
#    Отправь боту: /config
```

## ⚙️ Полный список возможностей

### 🔨 Модерация (20+ команд)

| Команда | Описание |
|---------|----------|
| `/mute @user [время]` | Замутить (5м/15м/30м/1ч/6ч/24ч) |
| `/unmute @user` | Снять мут |
| `/ban @user [причина]` | Забанить |
| `/unban @user` | Разбанить |
| `/kick @user [причина]` | Кикнуть |
| `/warn @user [причина]` | Варн (N варнов → автобан) |
| `/warns` | Проверить варны |
| `/warnlist @user` | Детальный список всех варнов |
| `/clearwarns @user` | Очистить варны пользователя |
| `/clear [N]` | Удалить N сообщений |
| `/del` | Быстро удалить replied-сообщение |
| `/pin` / `/unpin` | Закрепить / открепить |
| `/lock` | Закрыть чат (только админы пишут) |
| `/unlock` | Открыть чат |
| `/slowmode [N]` | Установить slowmode (сек) |
| `/mute_all` | Замутить всех участников |
| `/unmute_all` | Размутить всех |
| `/kickme` | Самоудаление из чата |
| `/admins` | Список администраторов |
| `/settitle` | Сменить название группы |
| `/setdescription` | Сменить описание группы |
| `/leave` | Бот покидает чат (только админ) |

### 👑 Админ-панель (/admin)

| Раздел | Возможности |
|--------|------------|
| 📊 Статистика | Пользователи, чаты, реклама (фильтр по своим чатам для chat admin) |
| 👑 Роли и админы | +/− Admin, +/− Moderator, списки (только SUPERADMIN) |
| 🚨 Жалобы | Просмотр и обработка жалоб (только SUPERADMIN/ADMIN) |
| 📢 Реклама | Заявки, одобрение/отклонение (фильтр по своим чатам для chat admin) |
| 👥 Пользователи | Список и информация (только SUPERADMIN/ADMIN) |
| ⚙️ Настройки чатов | Приветствие, ссылки, мат, капча, медиа, стикеры, цены, лог-канал, автоочистка |

#### 👑 Управление администраторами чатов

| Команда | Описание |
|---------|----------|
| `/setadmin <id> <чат1> [чат2...]` | Назначить администратора чата (SUPERADMIN) |
| `/removeadmin <id> <чат>` | Удалить администратора чата (SUPERADMIN) |
| `/chatadmins` | Список всех администраторов чатов |

Администратор чата назначается **автоматически** при добавлении бота в группу.
Он управляет настройками и рекламой **только своих чатов**.

### 🎁 Розыгрыши

| Команда | Описание |

## 📄 Конфигурация (.env)

Файл `.env` в корне проекта:

```ini
# Получить токен: @BotFather
BOT_TOKEN=1234567890:ABCdefGHIjklmNOPqrstUVwxyz

# Твой Telegram ID (узнать: @userinfobot)
ADMIN_IDS=[123456789]

# База данных (SQLite по умолчанию)
DATABASE_URL=sqlite+aiosqlite:///database.sqlite3

# Пропускать старые апдейты
SKIP_UPDATES=True
```

Команда `/config` покажет текущую конфигурацию (токен маскирован).
