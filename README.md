# 🤖 ModeratorBot

Мощный Telegram-бот на **aiogram 3** для модерации чатов, управления рекламой, автопостинга и администрирования.

## 🚀 Быстрый старт

```bash
# 1. Клонировать
git clone https://github.com/JamikKhidirov/ModeratorBot.git
cd ModeratorBot

# 2. Установить зависимости
pip install -r requirements.txt

# 3. Настроить .env
#    BOT_TOKEN=твой_токен
#    ADMIN_IDS=[твой_telegram_id]

# 4. Запустить
python app.py
```

## ⚙️ Полный список возможностей

### 🔨 Модерация (в группах)

| Команда | Описание |
|---------|----------|
| `/mute @user [время]` | Замутить (5м/15м/30м/1ч/6ч/24ч) |
| `/unmute @user` | Снять мут |
| `/ban @user [причина]` | Забанить |
| `/unban @user` | Разбанить |
| `/kick @user [причина]` | Кикнуть |
| `/warn @user [причина]` | Варн (N варнов → автобан) |
| `/warns` | Проверить варны |
| `/clear [N]` | Удалить сообщения |
| `/pin` / `/unpin` | Закрепить / открепить |

### 👑 Админ-панель (/admin)

| Раздел | Возможности |
|--------|------------|
| 📊 Статистика | Пользователи, чаты, реклама |
| 👑 Роли и админы | +/− Admin, +/− Moderator, списки |
| 🚨 Жалобы | Просмотр и обработка жалоб |
| 📢 Реклама | Заявки, одобрение/отклонение |
| 👥 Пользователи | Список и информация |
| ⚙️ Настройки чатов | Приветствие, ссылки, мат, капча, медиа, стикеры, лог-канал, автоочистка |

### 🤖 Автомодерация

| Фильтр | Описание |
|--------|---------|
| 🔗 Ссылки | Автоудаление + автоматический мут |
| 🤬 Мат | По списку слов (настраивается через UI) |
| 🎯 Триггеры | Слово → действие (удалить/мут/варн/бан) |
| 📸 Медиа | Удаление фото/видео (вкл/выкл) |
| 🎨 Стикеры | Удаление стикеров (вкл/выкл) |
| 🔒 Капча | Кнопочная капча для новых участников |
| 🛡️ Анти-рейд | Ограничение для аккаунтов младше N дней |
| 🔇 Тихий режим | Удаление без уведомления |

### 📢 Реклама и контент

| Возможность | Команда/Где |
|------------|------------|
| 💰 Купить рекламу | Меню "💰 Реклама" |
| ⭐ Пополнить баланс | `/buy` — оплата Telegram Stars |
| 📋 Мои объявления | Меню → "📋 Мои объявления" |
| 📊 Статистика | `/stat`, `/top`, `/mystat` |
| ⏰ Отложенные сообщения | `/schedule` |
| 📅 Регулярные посты | `/recurring_add` |
| 📡 RSS-автопостинг | `/addfeed <url>` |
| 🧹 Авто-очистка | Настройки чата → Автоочистка |
| 📝 Логирование в канал | Настройки чата → Лог-канал |

### 👋 Приветствия

Через **Визуальный редактор** (Настройки чата → Редактор):
- ✏️ Текст с переменными `{name}`, `{chat_title}`
- 🖼 Фото
- 🔘 Кнопки со ссылками
- 👀 Предпросмотр

## ⚠️ Требования

- Python 3.10+
- Бот должен быть **администратором** группы для модерации
- Для команд нужны права: ban, delete messages, restrict, pin
- Для лог-канала: бот должен быть админом в канале
